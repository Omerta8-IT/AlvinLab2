# Password Resets and Account Management

## Objective

Perform common Active Directory user account administration tasks that are routinely handled by an IT Help Desk technician.

## Environment

- Windows Server 2022 Domain Controller (DC02)
- Active Directory Users and Computers
- Windows 10 Domain Workstation

## Tasks Performed

- Created new user accounts
- Reset user passwords
- Unlocked user accounts
- Enabled and disabled user accounts
- Modified user account properties
- Assigned users to security groups
- Verified successful user logon after account changes

## Skills Demonstrated

- Active Directory Users and Computers
- User Account Administration
- Password Management
- Security Group Management
- Windows Server Administration
- IT Help Desk Support

## Outcome

Successfully managed Active Directory user accounts while following common enterprise IT support procedures for account creation, password management, and user administration.
