# Active Directory Domain Administration

## Objective

Build and administer a Windows Server Active Directory domain to simulate common enterprise IT support tasks.

## Environment

- Windows Server 2022 Domain Controller
- Windows Server File Server
- Windows 10 Domain Workstation
- VirtualBox
- Internal Virtual Network

## Tasks Performed

- Promoted Windows Server to a Domain Controller
- Created an Active Directory domain
- Created Organizational Units (OUs)
- Created security groups
- Created and managed user accounts
- Joined Windows 10 to the domain
- Reset user passwords
- Unlocked user accounts
- Added and removed users from security groups
- Verified domain authentication

## Skills Demonstrated

- Active Directory Users and Computers
- User Account Administration
- Group Management
- Password Management
- Windows Server Administration
- Windows 10 Administration
- IT Help Desk Troubleshooting

## Outcome

Successfully deployed and administered an Active Directory domain capable of supporting common enterprise IT support tasks and user management scenarios.
