# Shared Folder Access

## Objective

Configure shared folders on a Windows Server to allow secure network access for department users while restricting unauthorized access.

## Environment

- Windows Server 2022 File Server (FC01)
- Active Directory
- Department Security Groups
- Windows 10 Domain Workstation

## Tasks Performed

- Created shared folders for each department
- Configured network share permissions
- Published shared folders on the network
- Tested access using UNC paths
- Verified access from Windows 10 domain users
- Confirmed department users could access only their assigned shares
- Troubleshot shared folder connectivity

## Skills Demonstrated

- Shared Folder Configuration
- Share Permissions
- UNC Paths
- Windows File Sharing
- Active Directory Integration
- Windows Server Administration
- Troubleshooting

## Outcome

Successfully deployed department shared folders that allowed authorized users to securely access company resources across the network.
