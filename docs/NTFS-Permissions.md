# NTFS Permission Configuration

## Objective

Configure and troubleshoot NTFS permissions to provide users with the appropriate level of access while protecting sensitive company data.

## Environment

- Windows Server 2022
- Active Directory
- File Server (FC01)
- Domain User Accounts
- Security Groups

## Tasks Performed

- Created department shared folders
- Configured NTFS permissions
- Assigned permissions using security groups
- Verified least privilege access
- Tested Read, Modify, and Full Control permissions
- Removed unnecessary inherited permissions
- Verified user access from a Windows 10 workstation
- Troubleshot permission issues

## Skills Demonstrated

- NTFS Permissions
- Security Groups
- File Server Administration
- Least Privilege
- Access Control
- Windows Server Administration
- Troubleshooting

## Outcome

Successfully implemented secure NTFS permissions for departmental folders while ensuring users could access only the resources required for their job responsibilities.
